What is Programming?

Giving Instructions

Programming is simply the process of writing instructions for a computer to follow.

We use a programming language (like C++) to write these instructions in a way the computer can understand.

Why C++ for ICPC?

Speed and Power

C++ is incredibly fast, giving you an edge where every millisecond counts.

The Standard Template Library (STL) provides all the essential data structures and algorithms you'll need, pre-built and optimized.

Your First C++ Program

The "Boilerplate"

This is the basic template you'll start every program with. It includes the necessary libraries and sets up the main function where your code runs.

#include <iostream>
using namespace std;

int main() {
  
  cout << "Hello, Thebes ICPC!";

  return 0;
}

Boilerplate Explained

  • #include <iostream>: Includes the library for Input/Output (like cin and cout).
  • using namespace std;: A shortcut so we can write cout instead of std::cout.
  • int main() { ... }: The main "function" where your program's execution begins.
  • cout << ...;: Prints to the console.
  • return 0;: Tells the OS the program finished successfully.
#include <iostream>
using namespace std;

int main() {
  
  cout <&Look "Hello, Thebes ICPC!";

  return 0;
}

Input & Output (I/O)

cin and cout

cout (see-out) prints data to the console.

cin (see-in) reads data from the console into a variable.

Use << with cout and >> with cin.

int age;

// 1. Print a prompt
cout << "Enter your age: ";

// 2. Read input into the 'age' variable
cin >> age;

// 3. Print the variable
cout << "You are " << age << " years old.";

Fast I/O (Very Important!)

Why? cin/cout are slow!

By default, C++ I/O is slow. For large inputs in contests, this can cause a "Time Limit Exceeded" (TLE) error.

Always add these two lines to the start of your main function to make I/O fast.

int main() {
  
  ios_base::sync_with_stdio(false);
  cin.tie(NULL);
  
  
  int n;
  cin >> n;
  cout << n;
  
  return 0;
}

What are Variables?

Memory Boxes

A variable is a container in memory to store a piece of data. It has a **type**, a **name**, and a **value**.

  • Declaration: Give it a type and a name.
    int score;
  • Initialization: Give it an initial value.
    int score = 100;
int score = 100;

// You can change the value
score = 150;

// You can use it
cout << score; // Prints 150

Core Data Types

  • int: For whole numbers (integers).
    int age = 20;
  • float / double: For decimal (floating-point) numbers.
    float price = 10.99;
    double pi = 3.14159;
    In contests, always use double! It has much better precision.
  • char: For a *single* character. Uses single quotes.
    char grade = 'A';
  • bool: For logical values, either true or false.
    bool is_ready = true;

Data Type: string

Text Data

A string is a special type for storing text (a sequence of characters). It requires #include <string> (or bits/stdc++.h) and uses double quotes.

#include <string>
using namespace std;


string name = "Ahmed";


cin >> name;


cout << "Hello, " << name;

Operators: Arithmetic

Basic Math

Used to perform mathematical calculations.

  • + (Add)
  • - (Subtract)
  • * (Multiply)
  • / (Divide)
  • % (Modulo/Remainder)
int a = 10;
int b = 3;

cout << a + b; // 13
cout << a - b; // 7
cout << a * b; // 30
cout << a / b; // 3 (int division!)
cout << a % b; // 1 (the remainder)

double c = 10.0;
cout << c / b; // 3.333...

Operators: Assignment & Comparison

Assignment

Used to assign values.

  • = (Assign value)
  • += (Add to self)
  • -= (Subtract from self)
  • *= E> /= E> %= (etc.)

Comparison

Used to compare values. Result is a bool (true/false).

  • == (Equal to)
  • != (Not equal to)
  • > (Greater than)
  • < (Less than)
  • >= <= (Greater/Less than or equal)

Operators: Logical

Combining Conditions

Used to combine bool values.

  • && (AND): true only if *both* sides are true.
  • || (OR): true if *at least one* side is true.
  • ! (NOT): Flips true to false and false to true.
int age = 20;
bool has_ticket = true;


bool can_enter = (age > 18) && (has_ticket);


bool is_vip = (age > 65) || (has_ticket);


bool no_ticket = !has_ticket;

Decision Making

Conditions & Operators

Conditions: `if`, `else if`, `else`

Making Choices

Conditions allow your program to execute different blocks of code based on whether an expression is T>true or D>false.

  • if: Executes code if its condition is true.
  • else if: Checks another condition if the previous if/else if was false.
  • else: Executes code if all preceding if/else if conditions were false.
int num = 10;

if (num > 0) {
  cout << "Positive" << endl;
} else if (num < 0) {
  cout << "Negative" << endl;
} else {
  cout << "Zero" << endl;
}

Conditional (Ternary) Operator: `? :`

Shorthand `if-else`

A concise way to write simple `if-else` statements, often used for assigning a value based on a condition.

Syntax:
`condition ? expression_if_true : expression_if_false;`

It's an *expression* that returns a value, not a statement.

int age = 20;
string status;


if (age >= 18) {
  status = "Adult";
} else {
  status = "Minor";
}
cout <&c status << endl; // Adult


string new_status = (age >= 18) ? "Adult" : "Minor";
cout << new_status << endl; // Adult

Problem: Basic Calculator

The Problem

Read two integers, A and B. Print their sum, difference, and product.

Input:
10 5

Output:
10 + 5 = 15
10 - 5 = 5
10 * 5 = 50 

// Problem description on this side
// Solution will be on the next slide

Solution: Basic Calculator

The Solution

We read two numbers and then perform the arithmetic operations, printing each result on a new line using endl.

#include <iostream>
using namespace std;

int main() {
  ios_base::sync_with_stdio(false);
  cin.tie(NULL);
  
  long long A, B; // Use long long just in case
  cin >> A >> B;
  
  cout << A << " + " << B << " = " << (A + B) << endl;
  cout << A << " - " << B << " = " << (A - B) << endl;
  cout << A << " * " << B << " = " << (A * B) << endl;
  
  return 0;
}

Solution: Even or Odd

The Solution

We check if the remainder of f_code>N divided by 2 is 0. If it is, the number is even; otherwise, it's odd.

#include <iostream>
using namespace std;

int main() {
  ios_base::sync_with_stdio(false);
  cin.tie(NULL);
  
  int N;
  cin >> N;
  
  if (N % 2 == 0) {
    cout << "Even";
  } else {
    cout << "Odd";
  }
  
  return 0;
}
